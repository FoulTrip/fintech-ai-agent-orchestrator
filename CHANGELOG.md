# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-10-27

### Añadido
- Implementación inicial del **Orquestador de Agentes**.
- Módulo de **Paper Trading** con cálculo de P&L en tiempo real.
- Interfaz de usuario con **Next.js 14** y **shadcn/ui**.
- Integración de **Lightweight Charts** para visualización financiera.
- Backend en **FastAPI** con soporte para OpenAI/Anthropic y Mocks.
- Suite de pruebas con **Jest** y **React Testing Library**.
- Configuración de orquestación con **Docker Compose**.
