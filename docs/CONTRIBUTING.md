# Guía de Contribución

¡Gracias por querer contribuir a Fintech AI Agent Orchestrator!

## Estándares de Código

- **Clean Code & SOLID:** Mantén las funciones pequeñas y con una única responsabilidad.
- **Tipado:** El uso de TypeScript y Pydantic (en Python) es obligatorio.
- **Idioma:** Variables y documentación técnica en inglés. Comentarios explicativos en español.

## Proceso de Desarrollo

1. Haz un Fork del repositorio.
2. Crea una rama para tu feature (`git checkout -b feature/nueva-funcionalidad`).
3. Asegúrate de que todos los tests pasen (`npm test`).
4. Realiza un Pull Request detallando los cambios.

## Estructura de Carpetas

- `/frontend`: Aplicación Next.js.
- `/backend`: API FastAPI.
- `/docs`: Documentación técnica.
- `/__tests__`: Suite de pruebas.
